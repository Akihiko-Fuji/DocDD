# 用語集 / Glossary

- 文書ID: DOC-OVW-002
- 最終更新日: 2026-03-23

## 用語一覧

| 用語 | 定義 |
|---|---|
| A-TYPE | 本プロジェクトの主対象モード。ゲームオーバーになるまでプレイを続け、スコアと累計ライン数を伸ばす 1 人用モード |
| B-TYPE | 25 lines 消去を目標とする参考モード。一定本数のラインを消したら達成とみなす |
| Board | ブロックを積み上げる 10×18 の可視盤面。プレイヤーはこの枠内でピースを動かし、横一列をそろえて消去する |
| テトロミノ / Tetromino | テトリスで落下してくる、4 個の正方形ブロックで構成された図形の総称。I / O / T / J / L / S / Z の 7 種類を指す。一般的に「テトリミノ」と呼ばれることもある |
| Current Piece | 現在操作中の落下ピース。未固定のテトロミノであり、左右移動・回転・落下の対象となる |
| NEXT | 次に出現する 1 個先のピース表示。直後にどのテトロミノが来るかを示す |
| Spawn | 新しいテトロミノを既定位置に出現させること。通常は直前のピースが固定された後に行う |
| Lock | 現在のテトロミノを盤面上の固定済みブロックへ変換すること。固定後にライン消去判定や次ピース出現へ進む |
| Soft Drop | 下入力によって通常より速く落下させる操作。少しずつ位置を調整しながら早く下ろしたいときに使う |
| Hard Drop | テトロミノを一気に最下段まで落とす操作。本プロジェクトでは非採用 |
| Hold | 現在のテトロミノを一時保持し、あとで入れ替えて使う機能。本プロジェクトでは非採用 |
| ライン消去 / Line Clear | 盤面の横一列がすき間なく埋まったときに、その列を消去すること。テトリスの基本的な得点源であり、複数行を同時に消すほど高得点になりやすい |
| T-Spin | T テトロミノを回転操作で狭いくぼみに入れて固定し、所定条件を満たした状態。通常の落下だけでは入りにくい配置を回転で成立させる点が特徴で、本プロジェクトでは独自拡張として採用する |
| SELECT | Game Boy 系入力の 1 つ。PC では予約入力として扱う |
| START | 一時停止 / 再開や開始確定に用いる論理入力 |
