## 2. Product acceptance

### 2.1 初期受入の基準
- タイトルから A-TYPE を開始できる
- 10×18 盤面でプレイできる
- NEXT 1 個が管理される
- SELECT で NEXT 表示を ON/OFF できる
- Hold と Hard drop が存在しない
- ライン消去、得点、レベル進行が機能する
- Game Boy 基準の速度表が適用される
- T-Spin が仕様どおり判定される
- 一時停止とゲームオーバー後の導線が成立する

### 2.2 シナリオ別受入基準
#### 2.2.1 開始シナリオ
- 開始レベル 0〜9 を選択できる
- 開始確定後に盤面、SCORE、LINES、LEVEL、NEXT 表示状態が初期化される

#### 2.2.2 プレイシナリオ
- 左右移動、A/B 回転、ソフトドロップが有効である
- SELECT で NEXT 表示を切り替えられる
- 壁際・床際・積み上がり近傍で不正移動が抑止される

#### 2.2.3 得点シナリオ
- 通常ライン消去得点が得点表どおり加算される
- T-Spin 成立時に専用得点表が適用される
- A-TYPE のレベル進行と速度が計算式どおり反映される

#### 2.2.4 状態シナリオ
- START で一時停止、再度 START で再開できる
- ゲームオーバー後に再試行またはタイトル復帰を選べる
