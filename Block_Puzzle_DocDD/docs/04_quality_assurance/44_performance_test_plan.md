# 性能試験計画 / Performance Test Plan

- 文書ID: DOC-QA-044
- 最終更新日: 2026-03-24

## 1. 対象
- 入力応答性
- 長時間プレイ時の安定性
- 一時停止 / 再開後の進行安定性
- 固定更新幅の維持

## 2. 観測基準
- 入力反映: 通常は次 frame 以内
- pause / resume: 再開直後に同一押下で再 pause しない
- 長時間安定性: 30 分連続プレイ相当で進行不能がない
- 落下速度: `20_game_rules_spec.md` の速度表と矛盾しない
- フレーム基準: 60 fps / 16.67 ms 目標を前提とし、遅延超過はログで観測可能である

## 3. 実施方法
1. ヘッドレスまたはログ出力で frame 単位挙動を観測する
2. 代表レベル 0 / 9 / 15 / 20 で重力設定を確認する
3. START 同時押下、pause / resume、長時間連続プレイを確認する

## 4. 合格基準
- 同一試験環境（固定 step の開発実行環境）で各ケースを **3 回以上** 実施し、3 回とも観測基準を満たした場合に合格とする
- 入力応答性は各試行で 95% 以上の入力が次 frame 以内に反映され、遅延が 2 frame を超えないこと
- 長時間プレイ試験は 30 分相当を 3 回実施し、進行不能、クラッシュ、pause/resume 破綻が 0 件であること

## 5. 受入観点
- NFR-001, NFR-104, NFR-305 に対して観測方法が結びついていること
- 主観評価だけでなく frame または再現ログで確認できること
