## セミナーの主メッセージ（`01_script.md`準拠）

現場で実装を成功させるために、最初に明確化すべきは次の **3 文書**。

1. **何を正とみなすか**（データ契約）
2. **どこまでできたら完成か**（受入条件）
3. **誰がどう回すか**（運用ガードレール）

AI 活用の有無に関わらず、仕様の粒度を揃えることで
- 要件の合意がしやすくなる
- 検収がしやすくなる
- 実装のブレが減る

という効果を狙います。
