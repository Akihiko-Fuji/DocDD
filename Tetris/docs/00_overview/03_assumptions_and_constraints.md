# 前提条件と制約 / Assumptions and Constraints

- 文書ID: DOC-OVW-003
- 最終更新日: 2026-03-23

## 1. 前提
- 題材は公開可能な落下ブロックゲームとする
- 実装対象は PC とする
- 画像・キャラクタ素材は独自制作とする
- 操作体系は Game Boy 版準拠の論理入力を PC へ写像する

## 2. 制約
- 10×18 盤面を固定とする
- NEXT は 1 個のみとする
- Hold と Hard drop は採用しない
- A-TYPE を初期開発主軸とする
- B-TYPE は参考仕様とする
- T-Spin は独自拡張として採用する

## 3. 技術制約
- コードはコンパクトで見通し良く保つ
- 過剰なフレームワーク依存を避ける
- 文書とコードの対応づけやすさを優先する
