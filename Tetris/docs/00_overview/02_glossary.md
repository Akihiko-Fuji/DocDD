# 用語集 / Glossary

- 文書ID: DOC-OVW-002
- 最終更新日: 2026-03-23

## 用語一覧

| 用語 | 定義 |
|---|---|
| A-TYPE | 本プロジェクトの主対象モード。ゲームオーバーまでプレイし、スコアと累計ライン数を伸ばす |
| B-TYPE | 25 lines 消去を目標とする参考モード |
| Board | 10×18 の可視盤面 |
| Current Piece | 現在操作中の落下ピース |
| NEXT | 次に出現する 1 個先のピース表示 |
| Spawn | 現在ピースを既定位置に出現させること |
| Lock | 現在ピースを固定済みブロックへ変換すること |
| Soft Drop | 下入力による高速落下 |
| Hard Drop | 一気に最下段へ落とす操作。本プロジェクトでは非採用 |
| Hold | ピース保持機能。本プロジェクトでは非採用 |
| T-Spin | T ピースを回転で最終配置し、所定条件を満たした固定 |
| SELECT | Game Boy 系入力の 1 つ。PC では予約入力として扱う |
| START | 一時停止 / 再開や開始確定に用いる論理入力 |
