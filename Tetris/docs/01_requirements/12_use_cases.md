# ユースケース / Use Cases

- 文書ID: DOC-REQ-012
- 最終更新日: 2026-03-23

## UC-001 A-TYPE を開始する
- アクター: プレイヤー
- 事前条件: タイトル画面表示中
- 基本流れ:
  1. START 相当入力で開始設定へ進む
  2. レベルを選択する
  3. 開始を確定する
  4. プレイ画面へ遷移する

## UC-002 プレイ中にピースを操作する
- アクター: プレイヤー
- 基本流れ:
  1. 左右で位置調整する
  2. A/B で回転する
  3. 下でソフトドロップする
  4. ライン消去または T-Spin を狙う

## UC-003 一時停止する
- アクター: プレイヤー
- 基本流れ:
  1. プレイ中に START を押す
  2. PAUSE 表示を確認する
  3. START で再開する、またはタイトルへ戻る

## UC-004 ゲームオーバー後に再試行する
- アクター: プレイヤー
- 基本流れ:
  1. ゲームオーバー画面を確認する
  2. 再試行を選択する
  3. 開始設定へ戻る
