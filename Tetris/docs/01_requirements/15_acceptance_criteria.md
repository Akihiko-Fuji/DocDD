# 受入基準 / Acceptance Criteria

- 文書ID: DOC-REQ-015
- 最終更新日: 2026-03-23

## 1. 初期受入の基準
- タイトルから A-TYPE を開始できる
- 10×18 盤面でプレイできる
- NEXT 1 個が表示される
- Hold と Hard drop が存在しない
- ライン消去、得点、レベル進行が機能する
- T-Spin が仕様どおり判定される
- 一時停止とゲームオーバー後の導線が成立する

## 2. シナリオ別受入基準
### 2.1 開始シナリオ
- 開始レベル 0～9 を選択できる
- 開始確定後に盤面、SCORE、LINES、LEVEL が初期化される

### 2.2 プレイシナリオ
- 左右移動、A/B 回転、ソフトドロップが有効である
- 壁際・床際・積み上がり近傍で不正移動が抑止される
- NEXT 1 個が継続表示される

### 2.3 得点シナリオ
- 通常ライン消去得点が得点表どおり加算される
- T-Spin 成立時に専用得点表が適用される
- A-TYPE のレベル進行が計算式どおり反映される

### 2.4 状態シナリオ
- START で一時停止、再度 START で再開できる
- ゲームオーバー後に再試行またはタイトル復帰を選べる

## 3. 文書受入の基準
- 要求、仕様、設計、テストの役割が分離されている
- 主要仕様が追跡可能である
- A-TYPE 主軸 / B-TYPE 参考の境界が明確である
- T-Spin 採用の影響が上流から下流まで反映されている
