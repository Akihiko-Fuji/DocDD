## 1. 目的
プレイ中断と再開、ゲームオーバー後の振る舞いを外部仕様として定義する。

## 2. 一時停止
- プレイ中に START 相当入力を受けたとき一時停止する
- 一時停止中は自動落下、通常プレイ入力、主要ルール処理を停止する
- 盤面と数値情報は停止時点の内容を保持する
- 停止中に新しい現在ピース出現や得点更新が進行してはならない

## 3. 再開
- 一時停止中に START 相当入力を受けたときプレイへ戻る
- 再開時は停止前の状態を引き継ぐ
- 再開時に START 押しっぱなしで再度即時ポーズへ戻らないようにしてよい

## 4. タイトル復帰
- 一時停止中に B または ESC 相当入力でタイトルへ戻ってよい
- このとき進行中のプレイ状態は破棄してよい
- タイトル復帰時に途中スコアや盤面を引き継ぐ必要はない

## 5. ゲームオーバー
- 出現不能時にゲームオーバーとする
- ゲームオーバー中は通常プレイ入力を無効化する
- 最終 SCORE / LINES / LEVEL を表示する
- ゲームオーバー演出を採用する場合でも、再試行導線が隠れてはならない

## 6. 再試行
- ゲームオーバー画面で START または A により A-TYPE 開始設定へ戻る
- 再試行では新しいゲームセッションとして盤面・スコア・ライン数を再初期化する

## 7. 状態遷移表
| 現在状態 | 入力 / 条件 | 遷移先 | 備考 |
|---|---|---|---|
| プレイ中 | START | 一時停止 | 盤面状態を保持 |
| 一時停止 | START | プレイ中 | 停止前状態へ復帰 |
| 一時停止 | B / ESC | タイトル | セッション破棄可 |
| プレイ中 | 出現不能 | ゲームオーバー | 最終値を固定表示 |
| ゲームオーバー | START / A | 開始設定 | 再初期化して再試行 |
| ゲームオーバー | B | タイトル | セッション終了 |

## 8. 受入観点
- 一時停止中にピースが進行しないこと
- 再開後に停止前状態から継続すること
- ゲームオーバー後に通常プレイ入力が無効であること
- タイトル復帰と再試行の分岐が明確であること
